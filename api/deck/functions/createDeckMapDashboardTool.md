# Function: createDeckMapDashboardTool()

> **createDeckMapDashboardTool**(`params`): `Tool`

Creates a tool for adding Deck.gl map panels to dashboards.
Supports creating new map panels or updating existing ones with native Deck JSON configs.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `params` | `CreateDeckMapDashboardToolParams` | Parameters containing dashboard and database adapters |

## Returns

`Tool`

Tool instance for creating/updating Deck.gl map panels
