# Function: hslToHex()

> **hslToHex**(`h`, `s`, `l`): `string`

Converts HSL color values to Hex color string

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `h` | `number` | Hue (0-360) |
| `s` | `number` | Saturation (0-100) |
| `l` | `number` | Lightness (0-100) |

## Returns

`string`

Hex color string (#RRGGBB)
