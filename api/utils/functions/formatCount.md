# Function: formatCount()

> **formatCount**(`n`): `string`

Formats a number as a string with thousands separators and no decimal places.
Example: 1234567 -> "1,234,567"

## Parameters

| Parameter | Type |
| ------ | ------ |
| `n` | `number` | { `valueOf`: `number`; } |

## Returns

`string`
