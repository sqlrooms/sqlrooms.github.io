# Function: formatCount4()

> **formatCount4**(`n`): `string`

Formats a number as a string with 4 significant digits and SI prefix.
Example: 1234567 -> "1.235M"

| Parameter | Type |
| ------ | ------ |
| `n` | `number` | { `valueOf`: `number`; } |

## Returns
