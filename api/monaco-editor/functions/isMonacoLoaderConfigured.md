# Function: isMonacoLoaderConfigured()

> **isMonacoLoaderConfigured**(): `boolean`

Checks if the Monaco loader has been configured for bundling.
When true, Monaco is bundled with the app instead of loaded from CDN.

## Returns

`boolean`
