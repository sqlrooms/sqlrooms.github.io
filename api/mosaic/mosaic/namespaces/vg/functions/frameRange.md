# Function: frameRange()

> **frameRange**(`extent`, `exclude?`): `WindowFrameNode`

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `extent` | `FrameExtent` | The range-based window frame extent. |
| `exclude?` | `FrameExclude` | The window frame exclusion criteria. |

## Returns

`WindowFrameNode`

A window frame node.
