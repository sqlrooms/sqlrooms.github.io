# Function: frameRows()

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `extent` | `FrameExtent` | The row-based window frame extent. |
| `exclude?` | `FrameExclude` | The window frame exclusion criteria. |

## Returns
