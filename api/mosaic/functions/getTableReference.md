# Function: getTableReference()

> **getTableReference**(`table`): `string`

Returns the full table reference (database.schema.table) as a string,
without SQL escaping (for UI display and matching).

## Parameters

| Parameter | Type |
| ------ | ------ |
| `table` | `DataTable` |

## Returns

`string`
