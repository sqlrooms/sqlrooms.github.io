# Function: useMosaic()

> **useMosaic**(): `object`

Hook to manage the Mosaic connector.

## Returns

`object`

An object containing the Mosaic connector and a loading state.

| Name | Type | Default value |
| ------ | ------ | ------ |
|  `isMosaicLoading` | `boolean` | isLoading |
|  `mosaicConnector` | `undefined` | `Connector` | connector |
