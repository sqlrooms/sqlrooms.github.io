# Function: createDefaultAiToolRenderers()

> **createDefaultAiToolRenderers**(): [`DefaultAiToolRenderers`](../type-aliases/DefaultAiToolRenderers.md)

Creates the default tool renderer registry for the built-in AI tools.
Returns a typed map so it contributes to the `toolRenderers` constraint
when spread alongside custom renderers in [createAiSlice](createAiSlice.md).
